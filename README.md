# Global CO2 Emissions Analysis

## Overview
This project analyzes global CO2 emissions across industries and countries using SQL and Power BI.

## Objectives
- Identify top polluting industries
- Analyze country-wise emissions
- Visualize emission patterns

## Tools Used
- MySQL
- Power BI

## Key Insights
- Energy and manufacturing sectors dominate emissions
- Emissions are concentrated in major economies
- Industrial activity strongly influences emissions
